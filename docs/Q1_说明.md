## 三、代码设计

### 3.1 核心数据结构

```python
# 玩家枚举
class Player(enum.Enum):
    red = 1    # 红方先行
    black = 2  # 黑方后手

# 位置
Point = namedtuple('Point', ['row', 'col'])  # row: 0-9, col: 0-8

# 棋子
Piece = namedtuple('Piece', ['player', 'piece_type'])

# 棋子类型
class PieceType(enum.Enum):
    GENERAL, ADVISOR, ELEPHANT, HORSE, CHARIOT, CANNON, PAWN
```

## 四、随机AI实现

```python
class ChessRandomAgent:
    """随机选择合法走法"""

    def select_move(self, game_state: GameState) -> Move:
        legal_moves = game_state.legal_moves()
        return random.choice(legal_moves)
```

## 五、使用方式

### 5.1 Python API

```python
from cchess import GameState, Player
from agents.chess_random_agent import ChessRandomAgent

# 新游戏
game = GameState.new_game()

# 创建AI
agent = ChessRandomAgent()

# 对弈
while not game.is_over():
    move = agent.select_move(game)
    game = game.apply_move(move)

# 获取结果
winner = game.winner()
```
