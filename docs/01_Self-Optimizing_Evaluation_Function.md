# 中国象棋自优化评估函数研究

## 基本信息
- **标题**: Self-Optimizing Evaluation Function for Chinese-Chess
- **作者**: Xiangran Du, Min Zhang, Xizhao Wang
- **来源**: International Journal of Hybrid Information Technology, Vol.7, No.4 (2014), pp.163-172
- **关键词**: 人工智能, 中国象棋, 粒子群优化, 自学习, 评估函数

## 摘要

本文提出了一种新的优化中国象棋评估函数的方法——粒子群优化（PSO）。通过自优化方法自动调整评估函数中的参数，该方法通过竞争完成，即中国象棋系统使用不同的评估函数与自己对弈。实验结果表明，粒子群优化成功应用于中国象棋的评估函数优化，且经过多轮训练后程序性能得到有效提升。文章也通过比较有无地位控制的优化结果，考察了地位控制在评估函数中的重要性。

## 主要内容

### 1. 引言

- **背景**: 计算机象棋经过70多年发展，深蓝击败卡斯帕罗夫后，中国象棋成为新挑战
- **复杂度**: 中国象棋复杂度介于国际象棋和日本将棋之间
- **研究方向**: 评估函数优化和博弈系统智能化

### 2. 评估函数

#### 2.1 基本结构
```
Eval(x) = Eval(Red) - Eval(Black)  (if x = Red)
Eval(x) = Eval(Black) - Eval(Red)  (if x = Black)
```

#### 2.2 评估要素（6个主要方面）

1. **子力强度**
   - 将/帅: 无穷大（不可失去）
   - 标准子力价值比例: 车:马:炮:象:士:兵 = 9:4:4.5:2:2:1

2. **位置价值**
   - 不同位置对同一棋子的奖励/惩罚
   - 例: 兵过河价值大幅提升

3. **灵活度**
   - 棋子可行着法数
   - 体现攻防能力

4. **威胁与保护**
   - 棋子间相互威胁和保护关系
   - 体现配合程度

5. **地位控制**
   - 棋子可控制的位置
   - 在开/中局重要，残局价值降低

6. **棋子配合特征**
   - "连环马"、"马后炮"等特殊战术
   - 非简单叠加，体现协同作用

### 3. 粒子群优化(PSO)方法

#### 3.1 PSO基本原理
- **速度更新**: v(t+1) = w·v(t) + C1·rand·(LBest - x(t)) + C2·rand·(GBest - x(t))
- **位置更新**: x(t+1) = x(t) + v(t+1)
- **惯性权重**: w = (wmax - wmin) × (Tmax - t) / Tmax + wmin

#### 3.2 评估函数优化
- **粒子表示**: 每个粒子代表一组待优化参数（90个参数）
- **适应度函数**: 粒子群中不同粒子对弈的胜率
- **全局最优**: 最高胜率的粒子群中的参数

### 4. 实验结果

#### 第一阶段（400代，152000局对弈）
- 粒子群: 20个粒子，每个90个参数
- 结果: 第220代后性能明显进步，胜率从25%升至70%
- 问题: 优化后的评估函数在复杂局面判断不清，导致重复着法和无限循环

#### 第二阶段（1000代，380000局对弈）
- 参数调整: C1=C2=1.5, rand=0.3
- 结果: 全局最优从10%提升至70%
- 改进: 着法评估更智能，虽偶有误算但整体更优

#### 第三阶段（1000代）
- 参数: C1=C2=1, rand=0.1
- 结果: 对比前两阶段改进有限
- 发现: 残局防守存在严重问题

#### 4.2 对比结果
- **优化vs未优化**: 优化程序胜22局，8局平，0局负
- **地位控制影响**: 无地位控制的简化评估函数性能更优
  - 原因: 简化评估节省计算时间，使搜索深度更深

### 5. 结论
- PSO有效优化评估函数，程序能力明显进步
- 自动优化方法优于手工调试
- 地位控制在开/中局重要，残局可省略
- 残局防守仍需改进

## 关键发现
1. 粒子群优化通过自对弈逐步改进评估函数
2. 评估函数优化直接影响博弈系统整体性能
3. 简化评估函数可能比复杂函数更优（权衡速度和准确度）
4. 自动优化克服人工调参的主观性和耗时性
